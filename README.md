## Overview

This repository documents my **hands-on cybersecurity laboratory work** conducted in a **Kali Linux environment hosted on Apporto** as part of my **BSc (Hons) Information Technology studies**.

The objective of this lab series is to progressively develop **practical cybersecurity skills** starting from **Linux command line fundamentals** and moving toward **core security domains**, including:

- Linux system administration
- user account and permission management
- cryptography fundamentals
- network scanning and security
- vulnerability assessment
- cloud security principles
- penetration testing concepts

Each lab week contains **structured evidence** including:

- 📄 Lab reports (DOCX)
- 📷 Screenshots of terminal activity
- 📦 Archived lab artifacts
- 🧪 Demonstrations of commands and security tools

This repository serves both as:

- a **learning logbook**
- a **technical portfolio** demonstrating practical cybersecurity skills.
